# LocalTrade Application
Projet de marketplace composé de deux parties :
- Application Web
- Application Java Desktop

## Description
Cette application permet la gestion des utilisateurs , des produits et des feedbacks et des commandes  dans une marketplace.

## Technologies utilisées

### Web
- Symfony
- PHP
- MySQL
- HTML CSS JavaScript

### Java
- Java
- IntelliJ
- JDBC

## Fonctionnalités
- Inscription et connexion
- Gestion des produits , uder , fedback, commande
- Recherche
- Tableau de bord
- Mot de passe oublié : sms,email
